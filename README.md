# F'D Coffee Menu

Bu repository F'D Coffee kafesi uchun tayyorlangan **online menu** faylini o'z ichiga oladi.  
Mijozlar telefon orqali QR kod yordamida menyuni ko‘rib, buyurtma berishlari mumkin.

## Fayl tarkibi
- `index.html` — asosiy menyu fayli
- QR code — URL orqali menyuga kirish uchun

## Menu tarkibi
- **Coffee:** Espresso, Americano, Cappuccino, Latte, Raf Coffee, Signature Coffee  
- **Desserts:** Cheesecake, Chocolate Cake, Tiramisu, Croissant, Chocolate Croissant  
- **Sandwich:** Chicken Sandwich, Tuna Sandwich, Club Sandwich, Hot Dog  
- **Drinks:** Lemonade, Iced Tea, Mineral Water

## Qanday ishlatish
1. `index.html` faylini oching yoki GitHub Pages orqali saytga kirish.  
2. QR kodni skan qilgan holda menyuni telefonda ko‘ring.
